**1. 动作条件化的预测式缓存失效（closed-loop cache invalidation）。** VLA-Cache 是被动地比对相邻帧视觉 token 找静态区域。但你手里有一个别人没有的先验——**模型自己预测的动作**。机械臂/夹爪下一步要去哪，直接决定了图像里哪些区域会变。可以用预测的 action chunk 反过来预测哪些 visual token 会变 stale，做主动失效，而不是等下一帧来比对。

## 2. 核心假设

- **H1（可行性）** 帧间视觉变化的主要来源是机器人自身可预测的运动（夹爪/手臂占据区 + 头部相机自运动）；将其投影到图像平面得到的失效 mask，能覆盖绝大多数真正变化的 token。
- **H2（保真）** 在保持 prefix 长度不变、仅对 mask 内 token 重算 KV 的条件下，跨环境步复用其余 prefix KV 对成功率的损失可忽略（显著优于 CrossVLA 的全 prefix 复用）。
- **H3（人形增益）** 在头部相机会动的人形（G1）上，egomotion 补偿带来的增益明显大于静态相机设置——这是帧差类方法（VLA-Cache）失效的区间。
